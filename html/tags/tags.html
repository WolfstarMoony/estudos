Tags de Lista

  • Essa tag <ul> é usada para criar uma lista não ordenada.
  • Essa tag <ol> serve para criar uma lista ordenada.
  • Essa tag <li> serve para adicionar um item na lista.
  • Essa tag <dl> define uma lista de descrição.
  • Essa tag <dt> define um termo/nome em uma lista de descrição.
  • Essa tag <dd> é usada para descrever um termo/nome em uma lista de descrições.

Note que a tag <menu> foi removida, pois não é uma tag padrão HTML5 para listas.
