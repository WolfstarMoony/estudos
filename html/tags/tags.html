Tag de Imagem

A tag <img> é usada para inserir imagens. O atributo src define o caminho da imagem e alt fornece uma descrição alternativa.

Tags de Áudio e Vídeo

A tag <audio> é usada para incluir áudio em uma página. O atributo controls adiciona controles de reprodução.

A tag <video> é usada para incluir vídeos. O atributo controls adiciona controles de reprodução, e track é usado para legendas e outros textos.

Tag de Mapa e Área

A tag <map> define um mapa de imagem. A tag <area> define áreas clicáveis dentro do mapa.

Tag de Canvas

A tag <canvas> é usada para desenhar gráficos e animações em JavaScript. O elemento canvas deve ser manipulado com JavaScript para adicionar conteúdo.

Tag de Figure

A tag <figure> é usada para agrupar uma imagem e sua legenda. A tag <figcaption> fornece uma legenda para o conteúdo da tag <figure>.
