A tag <table> define uma tabela. As tags <tr> definem linhas da tabela, enquanto <td> define células de dados e <th> define células de cabeçalho. A tag <thead> agrupa o cabeçalho, <tbody> agrupa o corpo da tabela, e <tfoot> agrupa o rodapé. A tag <caption> fornece uma legenda para a tabela, e <col> pode ser usado para definir propriedades de colunas.
